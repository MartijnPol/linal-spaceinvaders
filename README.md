# Instructies

Onze SpaceInvaders is te besturen met de volgende toetsen:

- W (pitch up)
- A (roll left)
- S (pitch down)
- D (roll right)
- Q (yaw left)
- E (yaw right)
- H (assen tekenen waarom figuren draaien)
- 1 (front view)
- 2 (top view)
- 3 (side view)
- Shift (accelerate)
- Space (fire bullet)

Om het spel te starten, start simpelweg de SpaceInvaders.exe. Vervolgens kun je het spel spelen.
